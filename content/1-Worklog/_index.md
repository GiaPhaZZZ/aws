**Worklog Internship at FJC – 12-Week Overview**

On this page, I will introduce my worklog. I completed a 12-week internship at FJC, going through training and hands-on practice with AWS, from basic familiarity to deploying serverless and ML workflows on the cloud. Below is an overview of the weekly tasks:

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

* Interacted and got to know other members participating in FJC.
* Learned an overview of AWS and the services it provides.
* Opened an AWS account and practiced basic features.

**Week 2:** [Learning basic knowledge of EC2, Cloud9, S3](1.2-week2/)

* Mastered how to use EC2, Cloud9, and S3 to support AWS projects.
* Continued learning basic operations to build workflows.

**Week 3:** [Exploring Amazon Lightsail and scaling EC2](1.3-week3/)

* Learned the concept of Lightsail.
* Practiced scaling EC2 instances.

**Week 4:** [Grasping fundamental Cloud knowledge](1.4-week4/)

* Learned basic cloud concepts.
* Explored AWS CloudWatch, CloudFront, CLI, and management tools.

**Week 5:** [Migrate to AWS & optimize Lambda](1.5-week5/)

* Understood modules in “Migrate to AWS”.
* Learned how to optimize costs when using AWS Lambda.

**Week 6:** [AWS system management and lab practice](1.6-week6/)

* Learned about Grafana tags and managing AWS System Manager.
* Practiced lab exercises provided by FJC.

**Week 7:** [AWS CloudFormation & CDK](1.7-week7/)

* Studied AWS CloudFormation and CDK in detail.
* Learned to automate infrastructure and application deployment.

**Week 8:** [IaC, VPC Flow Logs, and Billing](1.8-week8/)

* Explored Infrastructure as Code (IaC) and VPC flow logs.
* Completed lab exercises related to billing and cost management.

**Week 9:** [Security and AWS SSO](1.9-week9/)

* Studied modules related to Security.
* Got familiar with AWS Single Sign-On and Security Hub.

**Week 10:** [Containerization & Deployment](1.10-week10/)

* Understood the process of containerizing applications with Docker.
* Deployed on AWS Cloud and got familiar with ECS, CDK, and CodePipeline.

**Week 11:** [AWS Serverless](1.11-week11/)

* Got familiar with AWS serverless services.
* Practiced with AWS SAM, Cognito, Lambda, and related services.

**Week 12:** [Data Lake & ML workflow](1.12-week12/)

* Understood the concept of Data Lake and its core components: S3, Glue, Athena, QuickSight.
* Practiced ingesting, processing, querying, and visualizing data on AWS.
* Got familiar with serverless analytics and machine learning workflows using SageMaker.
